== Description ==
Wordpress Plugin for GO Abroad HQ Clients. You will need your HQ Username and API Key to be able to use this plugin.

== Installation ==
- Download/Upload the GoAbroad HQ plugin to your site/blog
- Activate it 
- Then enter your [HQ Username] and [HQ API Key]
